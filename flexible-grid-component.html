CMS Control: Select Column Layout

CMS Control: Toggle Optional Links

How to implement in your CMS:

1. Add a dropdown field: "Column Layout" with options: 1, 2, 3, 4

2. Apply class to the container: cols-1, cols-2, cols-3, or cols-4

3. Each item needs: image, heading, summary (required), link (optional)

4. Works with any number of items - add as many as needed
