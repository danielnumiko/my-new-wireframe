Flexible Grid/List Component

Switch between layouts to see how the component adapts

CMS Control: Select Column Layout

CMS Control: Toggle Optional Links

How to implement in your CMS:

1. Add a dropdown field: "Column Layout" with options: 1, 2, 3, 4, 6

2. Apply class to the container: cols-1, cols-2, cols-3, cols-4, or cols-6

3. Each item has: image (mandatory), heading (mandatory), summary (optional), link (optional)

4. Works with any number of items - add as many as needed
